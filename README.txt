******DESCRIPTION******
This project takes in an address, a type of food, and a genre of movie and outputs the highest rated restaurant within that category,
directions to get there, and the highest rated movie as well as the nearest movie theater currently showing it within that genre.

APIS USED:
Google Maps
Yelp
IMBD/Rotten Tomatoes
